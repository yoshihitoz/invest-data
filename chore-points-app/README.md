# お手伝いポイント管理アプリ

義幸のお手伝いをポイント化し、親（Papa / Mama）が承認することでポイントが加算され、貯めたポイントを現金やゲーム時間に交換できるアプリです。

## 仕組み

1. 子ども（役割: `child`）がお手伝いの種類を選んで申請する
2. 親（役割: `parent`）が申請を承認 / 却下する。承認するとお手伝いの種類に設定されたポイントが加算される
3. 子どもは貯まったポイントを「現金」または「ゲーム時間」に交換申請できる（残高が足りない交換は申請できない）
4. 親が交換申請を承認すると、その分のポイントが差し引かれる

お手伝いの種類・ポイント数、交換メニュー・交換レートは初期値としてサンプル値を登録してあります。ログイン後、親のアカウントの「設定」タブからいつでも追加・変更できます。

## セットアップ

Node.js 22.5 以降が必要です（標準ライブラリの `node:sqlite` を使用するため）。

```bash
cd chore-points-app
npm install
SESSION_SECRET=<好きなランダム文字列> npm start
```

初回起動時にDB（`data/chore-points.db`、gitignore済み）が自動作成され、以下の初期ユーザーが登録されます。**必ずログイン後にパスワードを変更してください**（`POST /api/auth/change-password`、または今後UIから変更する運用にしてください）。

| username | 役割 | 仮パスワード |
|---|---|---|
| yoshihito | parent (Papa) | papa-change-me |
| masako | parent (Mama) | mama-change-me |
| yoshiyuki | child (義幸) | yoshiyuki-change-me |

`SESSION_SECRET` を設定しない場合、サーバー再起動のたびに全員がログアウトされます。運用時は固定の値を環境変数で設定してください。

## 初期データ（サンプル値・後で調整可能）

**お手伝いの種類**

| 種類 | ポイント |
|---|---|
| 皿洗い | 10pt |
| 洗濯物たたみ | 10pt |
| ゴミ出し | 5pt |
| 掃除機がけ | 15pt |
| お風呂掃除 | 20pt |
| 玄関掃除 | 15pt |

**交換メニュー**

| メニュー | 必要ポイント |
|---|---|
| 現金 100円 | 20pt |
| 現金 500円 | 90pt |
| ゲーム時間 15分 | 10pt |
| ゲーム時間 30分 | 18pt |

## 技術構成

- バックエンド: Node.js + Express + `node:sqlite`（Node標準の組み込みSQLite、追加のネイティブ依存なし）
- 認証: `express-session` によるセッション管理、パスワードは `crypto.scryptSync` でハッシュ化
- フロントエンド: ビルド不要のプレーンHTML/CSS/JS（`public/`）

## デプロイについて

このアプリは常駐プロセスが必要なため、GitHub Pagesのような静的ホスティングではなく、Node.jsを実行できる環境（家庭内サーバー、Fly.io、Render など）でホストしてください。`data/` ディレクトリを永続ボリュームにマウントすればポイント履歴が保持されます。
